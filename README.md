<h1>SOC Home Lab</h1>

<h2>Description</h2>
Project consists of building a simple SOC Lab at home with the help of the associated lesson at LetsDefend platform. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>PFSense</b> 
- <b>Windows 10 AD</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
- <b>Linux</b> (21H2)
